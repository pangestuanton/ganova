# Ganova - Klasik Tapi Beda Kasta

Website resmi **Ganova**, penyedia camilan **Gabin Fla** premium dengan berbagai varian rasa modern. Website ini dirancang untuk memudahkan pelanggan dalam menjelajahi menu dan melakukan pemesanan langsung melalui WhatsApp.

## 🚀 Cara Penggunaan Website

Website ini adalah *Single Page Application* (satu halaman) yang sangat mudah digunakan:

### 1. Navigasi Halaman
- Gunakan menu di bagian atas (Header) untuk berpindah antar bagian: **Beranda**, **Tentang Kami**, **Menu**, **Lokasi**, dan **Kontak**.
- Di perangkat mobile, klik ikon garis tiga di pojok kanan atas untuk membuka menu navigasi.

### 2. Melihat Produk & Menu
- Scroll ke bagian **Menu Kami** untuk melihat berbagai varian Gafla (Gabin Fla).
- Setiap kartu produk menampilkan nama varian, deskripsi rasa, dan harga terbaru (Rp 12.000).

### 3. Melakukan Pemesanan
- Klik tombol **"Pesan Sekarang"** di bagian Hero (atas) atau tombol **"Pesan via WhatsApp"** di setiap kartu menu.
- Anda akan diarahkan secara otomatis ke bagian **Hubungi Kami**.
- Pilih salah satu admin (Admin 1 atau Admin 2) untuk memulai chat WhatsApp dan melakukan pemesanan.

### 4. Melihat Lokasi & Jam Operasional
- Klik menu **Lokasi** di navigasi atau footer.
- Sebuah kotak (modal) akan muncul menampilkan alamat lengkap di **Jl. Poros Bawah, Gisting Bawah, Kec. Gisting, Kabupaten Tanggamus, Lampung 35384**.
- Klik tombol **"Lihat di Google Maps"** untuk melihat rute atau titik lokasi di aplikasi peta.

### 5. Informasi Hukum & Ketentuan
- Klik link **Syarat & Ketentuan** atau **Kebijakan Privasi** di bagian footer (bawah).
- Informasi akan muncul dalam bentuk kotak (modal) tanpa harus berpindah halaman.

---

## 🛠️ Struktur Proyek
- `index.html`: Struktur utama halaman web dan modal.
- `css/styles.css`: Desain visual, tata letak, dan animasi.
- `js/script.js`: Fungsi interaksi (menu mobile, modal, dan smooth scroll).
- `assets/`: Folder berisi gambar produk dan logo brand.
