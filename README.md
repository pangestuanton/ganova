# Ganova - Klasik Tapi Beda Kasta

Website resmi **Ganova**, penyedia camilan **Gabin Fla** premium dengan berbagai varian rasa modern.

## Fitur Utama
- **Landing Page Interaktif**: Menampilkan produk dan keunggulan brand.
- **Sistem Pemesanan WhatsApp**: Memudahkan pelanggan memesan langsung ke admin.
- **Informasi Lokasi & COD**: Mendukung pengiriman area Gisting, Tanggamus.
- **Dokumentasi Hukum**: Dilengkapi Syarat & Ketentuan serta Kebijakan Privasi dalam bentuk modal.

- `js/script.js`: Logika interaksi dan modal.
- `assets/`: Aset gambar dan logo.
